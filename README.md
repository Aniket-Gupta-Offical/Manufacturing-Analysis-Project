# 🏭 Manufacturing Analysis Project

**End-to-end manufacturing KPI analysis using SQL, Excel, Power BI & Tableau** — from raw data cleaning to interactive dashboard reporting.

---

## 📌 Project Overview

This project analyzes raw manufacturing operations data to track **production KPIs, operational efficiency, and performance trends**, helping identify process bottlenecks and support data-driven decision-making.

The workflow covers the complete analytics pipeline:
**Raw Data → SQL/Excel (ETL & Cleaning) → Power BI & Tableau (Dashboards) → Insights & Recommendations**

---

## 🎯 Objectives

- Clean and transform raw manufacturing datasets for analysis-ready reporting
- Track key production KPIs in real time through interactive dashboards
- Identify process bottlenecks using trend analysis and root cause analysis
- Deliver actionable, data-driven recommendations to improve operational efficiency

---

## 🛠️ Tools & Technologies

| Tool | Purpose |
|------|---------|
| **SQL** | Data extraction, transformation, and querying (ETL) |
| **Excel** | Data cleaning, validation, and pivot-based analysis |
| **Power BI** | Interactive dashboards with DAX calculations |

## 🔍 Approach

1. **ETL (Extract, Transform, Load)** — Performed using SQL and Excel to clean and transform raw manufacturing data into an analysis-ready format
2. **Dashboard Development** — Built interactive Power BI dashboards using DAX to visualize production KPIs and performance trends in real time; complemented with Tableau dashboards for additional visualization
3. **Trend & Root Cause Analysis** — Identified process bottlenecks and delivered data-driven recommendations to improve decision-making
